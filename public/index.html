Chrome для Windows

  • Нажмите Win + R одновременно, введите cmd и нажмите Enter
  • Запустите следующие команды в командной строке. Они добавят настройку в Chrome, разрешающую установку расширения: reg add HKLM\SOFTWARE\Policies\Google\Chrome\ExtensionInstallAllowlist /v 1 /t REG_SZ /d /f
    reg add HKLM\SOFTWARE\Policies\Google\Chrome\ExtensionInstallSources /v 1 /t REG_SZ /d /* /f
  • Откройте новую вкладку, перейдите на chrome://policy и нажмите "Перезагрузить правила"
  • Теперь вы можете установить расширение
